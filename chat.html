D1F Chat

Multi-user serverless peer-to-peer chat powered by WebRTC

🚀 How to Use

  1. Enter your username: Choose a display name for the chat
  2. Create or join a room: Enter a room name (all participants must use the same name)
  3. Share the room name: Give the room name to everyone you want to chat with
  4. Connect: All participants click "Connect" - connections happen automatically
  5. Multi-user chat: Support for multiple people in the same room! No servers, completely peer-to-peer
